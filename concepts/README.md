# 📚 Concepts

This folder contains theoretical explanations of important Android and Kotlin concepts.

It is designed as a quick reference and revision guide, along with simple examples to reinforce understanding.

---

## 🎯 Purpose

* Build strong fundamentals
* Serve as a revision guide for interviews
* Provide concise explanations with examples
* Complement projects and code snippets

---

## 🧠 Topics Covered

* Kotlin Basics
* Coroutines
* Flow / StateFlow
* Retrofit
* Room Database
* MVVM Architecture
* Dependency Injection (Hilt)
* Error Handling & API States

---

## 📂 Structure

Each concept is organized either as:

* A single markdown file

  ```plaintext
  concepts/room.md
  ```

* Or a folder with detailed notes

  ```plaintext
  concepts/coroutines/
      README.md
  ```

---

## ✍️ Format of Each Concept

Every concept file follows this structure:

* What is it?
* Why is it used?
* Key components
* Simple example
* When to use / Best practices
